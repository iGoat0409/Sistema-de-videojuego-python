# 🎮 Sistema de Videojuegos en Python

Un sistema integral de juegos interactivos desarrollado en Python, que ofrece múltiples opciones de entretenimiento con seguimiento de estadísticas en tiempo real.

[![Python Version](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)

---

## 📝 Descripción

Sistema modular de juegos de entretenimiento desarrollado como proyecto académico. Incluye una interfaz interactiva por línea de comandos que permite a los usuarios acceder a diferentes juegos, rastrear sus estadísticas personales y disfrutar de una experiencia de juego completa.



## ✨ Características

- 🎲 **4 Juegos Diferentes**: Mayor o Menor, Número Secreto, Par o Impar y Blackjack
- 📊 **Sistema de Estadísticas**: Seguimiento completo de puntuaciones, rachas y victorias
- 💳 **Sistema de Créditos**: Gestión de recursos en juegos de apuesta
- 🔄 **Interfaz Interactiva**: Menú principal con navegación intuitiva
- ⚡ **Validación de Entrada**: Manejo robusto de entradas del usuario
- 📈 **Reportes Generales**: Visualización de estadísticas acumuladas por jugador

## 🎯 Juegos Disponibles

### A) Mayor o Menor
**Descripción**: Adivina si el siguiente número aleatorio será mayor o menor que el actual.

**Mecánica**:
- Se genera un número inicial (1-1000)
- Debes predecir si el siguiente será mayor o menor
- Cada acierto incrementa tu racha
- El juego termina en el primer error

**Objetivo**: Alcanzar la mayor racha de aciertos posible

---

### B) Número Secreto
**Descripción**: Adivina un número secreto del 1 al 100 en máximo 6 intentos.

**Mecánica**:
- El sistema genera un número aleatorio (1-100)
- Tienes 6 intentos para adivinarlo
- Recibirás pistas ("es mayor" o "es menor")
- El juego indica si ganaste o perdiste

**Objetivo**: Adivinar el número en el menor número de intentos

---

### C) Blackjack
**Estado**: 🚧 En construcción

Este juego estará disponible en próximas versiones.

---

### D) Par o Impar
**Descripción**: Apuesta créditos prediciendo si la suma de dos dados será par o impar.

**Mecánica**:
- Comienzas con 100 créditos
- Apuestas una cantidad de créditos
- Se lanzan dos dados (1-6 cada uno)
- Predices si la suma será PAR o IMPAR
- Ganas/pierdes tu apuesta según el resultado

**Objetivo**: Maximizar tus créditos antes de perderlos todos

---

### E) Reporte de Estadísticas
Visualiza las estadísticas acumuladas de todos los juegos:
- Último jugador en cada juego
- Cantidad de veces jugado
- Victorias y derrotas
- Mejores rachas alcanzadas
